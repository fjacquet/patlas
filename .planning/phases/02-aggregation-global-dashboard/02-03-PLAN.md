## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| EstateView (workbook-derived) → DOM | VM names/hostnames/aggregates flow into rendered UI; an error path that interpolates a row/error object would leak PII (Critical-2) |
| user → accounting-mode toggle | The only interactive input on the dashboard; a UI selection, not a dataset row (PAR-05 unaffected — in-memory state only) |
| new echarts deps → rendered SVG | Render-only; Phase-1 fetch/XHR/WS/Beacon guard + CSP `connect-src 'self'` still active |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-09 | Information disclosure | GlobalDashboard inline error state | mitigate | Renders `error.message`/`error.name` ONLY, never the error/Snapshot/VInfoRow object (mirrors FallbackError.tsx); the Phase-1 ErrorBoundary still wraps App as the outer net |
