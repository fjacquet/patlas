<objective>
Port + extend vsizer's RVTools parser into a Web Worker, behind a typed `postMessage` boundary, that returns a fully-validated `Snapshot` payload — handles RVTools 3.10/3.11/4.0/4.4 column drift, infers `capturedAt`/`vCenterLabel`/`rvtoolsVersion`, captures `viSdkUuid` for Phase 4, drops the SheetJS WorkBook reference inside the worker (memory budget), and surfaces structured errors naming missing sheets/columns.

Purpose: PAR-01 (Worker boundary), PAR-02 (clear errors), PAR-03 (alias dictionary), PAR-04 (MiB regression via the canary) are all one cohesive subsystem. Splitting them across plans would require the same files to be touched in multiple plans, breaking the file-ownership rule. This plan owns the entire parser surface and ships it pure (`engines/parser/**`) — no React, no DOM, no Zustand imports.

Output: A `parseInWorker(file)` function the main thread calls to get a `Snapshot`. The worker runs SheetJS off the main thread, the `WorkBook` reference is dropped at the worker boundary, the response is a canonical typed payload only, and the MiB canary fixture proves PAR-04 is locked in.
</objective>

```typescript
import type { ParsedWorkbook } from './parseXlsx'
import type { VInfoRow } from '@/types/vinfo'

const ISO_DATE_RE = /(\d{4})-(\d{2})-(\d{2})(?:[_T](\d{2})[.:](\d{2})[.:](\d{2}))?/

export const inferCaptureDate = (
  filename: string,
  mtime: number,
  sheets: ParsedWorkbook,
  explicit?: Date,
): Date => {
  if (explicit) return explicit
  const m = filename.match(ISO_DATE_RE)
  if (m) {
    const [, y, mo, d, h, mi, s] = m
    const iso = `${y}-${mo}-${d}T${h ?? '00'}:${mi ?? '00'}:${s ?? '00'}Z`
    const parsed = new Date(iso)
    if (!Number.isNaN(parsed.getTime())) return parsed
  }
  const meta = sheets.sheets.get('vMetaData') ?? sheets.sheets.get('vmetadata')
  if (meta && meta.rows.length > 0) {
    // vMetaData is shaped as Property/Value rows
    const cell = meta.rows.find((r) =>
      String(r.Property ?? r.property ?? '').toLowerCase().includes('exported'))?.Value
        ?? meta.rows[0]?.['Exported Timestamp']
    if (cell != null) {
      const d = new Date(String(cell))
      if (!Number.isNaN(d.getTime())) return d
    }
  }
  return new Date(mtime)
}

export const inferVCenterLabel = (vinfo: VInfoRow[], filename: string): string => {
  const fromRow = vinfo.find((r) => r.viSdkServer.trim().length > 0)?.viSdkServer.trim()
  if (fromRow) return fromRow
  // Filename stem fallback: 'RVTools_export_all_2026-01-07_MOM-vCenter.xlsx' → 'MOM-vCenter'
  const stem = filename.replace(/\.xlsx$/i, '').replace(/^RVTools_export_(all|hosts|vms)_/, '')
  return stem || filename
}

export const inferRvtoolsVersion = (sheets: ParsedWorkbook): string => {
  const meta = sheets.sheets.get('vMetaData') ?? sheets.sheets.get('vmetadata')
  if (meta) {
    const versionCell = meta.rows.find((r) =>
      String(r.Property ?? r.property ?? '').toLowerCase().includes('rvtools'))?.Value
    if (versionCell != null) {
      const s = String(versionCell).trim()
      if (s) return s
    }
  }
  // Marker-column sniffing fallback (per RESEARCH.md Pattern 3 / Pitfall 5):
  const vinfo = sheets.sheets.get('vInfo') ?? sheets.sheets.get('vinfo')
  if (vinfo) {
    const headers = (vinfo.headers ?? []).map((h) => h.toLowerCase())
    if (headers.includes('virtual machine tags')) return '4.0+'
    if (headers.includes('creation date')) return '3.11+'
  }
  return 'unknown'
}
```

`src/engines/parser/parseInWorker.ts` — per RESEARCH.md Pattern 1 (L425-456) and PATTERNS.md L735-742:
```typescript
import type { Snapshot, ParseError } from '@/types/snapshot'

type ParseRequest = { kind: 'parse'; buf: ArrayBuffer; filename: string; mtime: number }
type ParseResponse =
  | { kind: 'ok'; snapshot: Omit<Snapshot, 'id' | 'parsedAt'>; warnings: ParseError[] }
  | { kind: 'err'; error: { name: string; message: string; sheet?: string; column?: string; kind?: string } }

let worker: Worker | null = null

const getWorker = (): Worker => {
  if (!worker) {
    worker = new Worker(new URL('./parser.worker.ts', import.meta.url), { type: 'module' })
  }
  return worker
}

export const parseInWorker = async (
  file: File,
): Promise<{ snapshot: Omit<Snapshot, 'id' | 'parsedAt'>; warnings: ParseError[] }> => {
  const w = getWorker()
  const buf = await file.arrayBuffer()
  return new Promise((resolve, reject) => {
    const onMessage = (e: MessageEvent<ParseResponse>) => {
      w.removeEventListener('message', onMessage)
      if (e.data.kind === 'ok') resolve({ snapshot: e.data.snapshot, warnings: e.data.warnings })
      else reject(Object.assign(new Error(e.data.error.message), e.data.error))
    }
    w.addEventListener('message', onMessage)
    const msg: ParseRequest = { kind: 'parse', buf, filename: file.name, mtime: file.lastModified }
    w.postMessage(msg, [buf]) // transferable — buf is neutered on the main thread after this
  })
}
```

## Trust Boundaries

|----------|-------------|
| File → ArrayBuffer (main thread) | Untrusted `.xlsx` bytes arrive from the user. The main thread only forwards the buffer (transferable) into the worker; no parsing on the main thread. |
